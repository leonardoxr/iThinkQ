# Privacy Notes

iThinkQ controls home appliances, so privacy is part of the product surface.

## Stored Locally

- ThinQ Personal Access Token in macOS Keychain
- Country, client ID, preferences, favorites, aliases, and visual choices in app preferences
- Device/profile/status cache under Application Support

## Not Intentionally Logged

- Personal Access Tokens
- Raw MQTT topics and payloads
- Client certificates or private keys
- Raw personal device data

## Sharing Debug Data

Before sharing logs, screenshots, or fixtures:

- Remove device IDs
- Remove account IDs
- Remove room/home names
- Remove raw MQTT topics and payloads
- Remove addresses, emails, and tokens

When in doubt, do not post it publicly.
