# Roadmap

ThinkQ is heavily WIP. This roadmap is intentionally practical and may change as real-device behavior is discovered.

## Near Term

- Improve profile decoding against more real LG devices.
- Add safe sanitized fixtures for each device family.
- Expand bespoke controls beyond air conditioners.
- Improve menu bar-only mode and quick controls.
- Add clearer error states for offline devices and rate limits.
- Automate prerelease packaging from GitHub tags.

## Mid Term

- Harden MQTT reconnect and client certificate lifecycle.
- Add notification preferences for cycles, offline devices, and errors.
- Add safer command previews and high-risk confirmations.
- Improve localization.
- Add app-level cache reset and privacy tools.

## Later

- Signed and notarized distribution.
- Installer or auto-update strategy.
- More polished icon/screenshot/release assets.
