# Changelog

## 0.1.0-wip

Initial public WIP release.

- Native SwiftUI macOS app shell with full window and menu bar modes.
- Onboarding for ThinQ Personal Access Token setup and token testing.
- Keychain token storage.
- ThinQ device discovery, profile parsing, status refresh, and control validation.
- Bespoke air-conditioner controls and generic profile-driven fallback controls.
- Device customization persistence for favorites and aliases.
- Local cache, polling, and rate-limit backoff.
- MQTT live-event connection groundwork.
- WIP glass-style UI surfaces, menu bar dashboard, settings, and app icon.
